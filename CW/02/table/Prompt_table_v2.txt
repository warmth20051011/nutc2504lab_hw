# Task

# Rules
1. 問題必須模擬使用者在查詢資料庫時的真實口吻。
2. 答案必須完全依據表格內容，不可自行捏造。
3. 涵蓋不同維度：包含單點查詢（例如：某某的價格是多少？）與比較查詢（例如：哪個產品最貴？）。
4. 輸出格式為 JSON，包含 "question" 和 "answer" 欄位。
